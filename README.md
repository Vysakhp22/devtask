# devtask
DevTask is a personal sprint and Kanban tracker built to explore Angular 22 features, Spring Boot APIs, Docker, GitHub Actions CI/CD, and AWS deployment.


# DevTask

Personal Sprint & Kanban Tracker

## Tech Stack

### Frontend
- Angular 22
- Signals
- Signal Forms
- Tailwind CSS v4

### Backend
- Spring Boot 3
- Spring Data JPA
- PostgreSQL

### DevOps
- Docker
- GitHub Actions

### AWS
- S3
- CloudFront
- ECS Fargate
- ECR

## Goals

- Learn Angular 22 features
- Learn Spring Boot
- Learn AWS ECS deployment
- Learn GitHub Actions CI/CD
